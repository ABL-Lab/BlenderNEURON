Required Software
=========================================

BlenderNEURON has the following software requirements:


 1. OS: Windows (tested with 10), Mac OS (tested with Mac OS 10.12), or Linux (tested with Ubuntu 16.04)
 2. Working installation of `NEURON <https://neuron.yale.edu>`_ (tested with v7.5 standard HOC and compiled Python installations)
 3. Working installation of `Blender <https://www.blender.org/download/>`_ (tested with 2.7.9)
 4. To use the Python library: Either NEURON launched with the `-python` flag OR a working `compiled NEURON+Python installation <https://neuron.yale.edu/neuron/getstd>`_ (e.g. `from neuron import h` works from Python) (tested with Python 2.7 and 3.5)
